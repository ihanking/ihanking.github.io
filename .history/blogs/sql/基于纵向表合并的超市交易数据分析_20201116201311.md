三表合并
```sql
SELECT
	shop_id,uid,order_id,date,amt3
FROM
	TransA1710
WHERE
	pay_type = 1
UNION ALL
SELECT
	shop_id,uid,order_id,date,amt3
FROM
	TransA1801
WHERE
	pay_type = 1
UNION ALL
SELECT
	shop_id,uid,order_id,date,amt3
FROM
	TransB1805
WHERE
	pay_type = 1;
```
