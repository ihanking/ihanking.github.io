**ON关键词后面跟不等式**
```sql
select
    t1.*,
    t2.Python,
    t2.Visualization
from
    stu_info as t1
left join
    stu_score as t2
on
    t1.id=t2.id
and
    MySQL > 90;
```

```sql
select
    t1.*,
    t2.Python,
    t2.Visualization
from
    stu_info as t1
left join
    stu_score as t2
on t1.id=t2.id 
where
    MySQL > 90;
```
