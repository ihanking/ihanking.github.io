### 唯一索引

**无索引查询**
```sql
SELECT *
FROM tourism_orders
WHERE userid = '100000001445';
```

**基于唯一索引查询**
```sql
#创建索引
create UNIQUE index id_idx on tourism_orders(userid,orderid);

select *
from tourism_orders
where userid='100000001445';
```
