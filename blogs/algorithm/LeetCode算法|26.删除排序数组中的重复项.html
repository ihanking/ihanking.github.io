LeetCode算法|26.删除排序数组中的重复项

Hank 2020-09-13 leetcode
class Solution:
    def removeDuplicates(self, nums) -> int:
        nums[:] = list(sorted(set(nums)))
        return len(nums)

nums=[0,0,1,1,1,2,2,3,3,4]
result=Solution().removeDuplicates(nums)
print(result)
