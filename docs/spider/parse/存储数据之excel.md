* 第一种方式是使用pandas库
```
def save(data):
    data = pd.DataFrame(data)
    data.to_excel(index=False)
```

* 第二种方式是使用openpyxl库
```
import openpyxl
def save(title, head, data):
    outwb = openpyxl.Workbook()
    outws = outwb.active
    outws.title=title
    for h in range(len(head)):
        outws.cell(1,h+1).value=head[h]
    for row in data:
        outws.append(row)
    outwb.save("path")
```

* 第三种方式是使用xlwt库

def save(title, head, data):
